# Analysis prose deviations from legacy comments

This document records places where the `analysis/` notes intentionally deviate from the legacy comments while preserving methodological intent. The default rule is to keep legacy prose and order wherever possible, replace copied legacy numbers with current target-backed values, and document any substantive wording changes.

Remaining known limitation: this document records major prose deviations, not every punctuation or wording adjustment. Future changes to `analysis/` should either keep legacy prose unchanged or add a row here when wording changes for methodological or architectural reasons.
