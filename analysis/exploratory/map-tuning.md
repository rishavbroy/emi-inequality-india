## Legacy diagnostic intent

The legacy map-making chunk used commented exploratory code for palette
selection, break checks, `cols4all`/GUI inspection, `tmap_save()` sizing
experiments, and k-means/break exploration for the EMI and
linguistic-distance maps. The current production maps are generated by
the `figure_files` target; this note keeps the tuning rationale in
`analysis/` and renders the current figure artifacts instead of copying
static screenshots or comments.
